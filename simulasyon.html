Kullanılan Ücretsiz API'ler
  • Nominatim — Mahalle → koordinat (OpenStreetMap)
  • Overpass API — Gerçek toplanma alanları / parklar
  • Open-Elevation — Waypoint Z değerleri (SRTM)
  • CartoDB Dark — Harita tile'ları
  • Qwen2.5-VL-7B — Görüntü analizi (MLX / localhost)
